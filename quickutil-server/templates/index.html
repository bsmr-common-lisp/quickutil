What's Quickutil?

Quickutil is an application and library for distributing Common Lisp utilities, aiming to solve the "utility problem".

It is similar in spirit to Quicklisp, in that utilities can automatically be downloaded and installed to projects.

Installation

(ql:quickload :quickutil-client)

Load an Utility

(quickutil-client:quickload :iota)
